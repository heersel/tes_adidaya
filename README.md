# tes_adidaya
ini adalah laravel livewire yg digunakan hendri purnianto untuk tes adidaya

tata cara menjalankan aplikasi
1. deploy file
2. jalankan composer dump-autoload
3. koneksikan DBnya seperti IP, port dan nama DB di file .env
4. jalankan php artisan migrate
5. php artisan key:generate

untuk menggunakan aplikasi, karena aplikasi sederhana maka form inventaris langsung saya tujuan di url root aplikasi
